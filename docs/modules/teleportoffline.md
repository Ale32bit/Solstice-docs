# Teleport Offline

This module adds the `/tpoffline` command to teleport you to logoff position of a player.

## Commands

!!! command "tpoffline &lt;player&gt;"

    Teleport to the logoff position of the player.

    **Permissions**

    * `solstice.teleportoffline.base` - Default: 2

    **Aliases**

    * tpoff
