# Home

The home module brings player home waypoints.

## Configuration

### Max Homes - `maxHomes`

Maximum amount of homes a player can have.

Set to `-1` to disable.

## Commands

- `/home <name>`
- `/sethome <name> [force]`
- `/delhome <name>`
- `/homes`
- `/homes <player>` - (`solstice.command.home.others`)
- `/homeother <player> <name>`
