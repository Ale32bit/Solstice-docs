# Mute

This module allows staff members to mute players and prevent sending chat messages.

## Commands

!!! command "mute &lt;targets&gt;"

    Mute players.

    **Permissions**

    * `solstice.mute.base` - Default: 2

!!! command "unmute &lt;targets&gt;"

    Unmute players.

    **Permissions**

    * `solstice.mute.base` - Default: 2
